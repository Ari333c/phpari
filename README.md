# phpari
<img width="1845" height="1001" alt="image" src="https://github.com/user-attachments/assets/03c61eac-30e1-44ef-b504-71387afa3be8" />
 Описание
Консольное приложение на PHP, имитирующее разговор с глухой бабушкой.

## Задание 1 (Базовая версия)
- Если фраза не заканчивается "!" → ответ: "АСЬ?! ГОВОРИ ГРОМЧЕ, ВНУЧЕК!"
- Если фраза заканчивается "!" → ответ: "НЕТ, НИ РАЗУ С {random year} ГОДА!"
- Для выхода нужно сказать "ПОКА!"

## Задание 2 (Улучшенная версия, )
- Нужно сказать "ПОКА!" три раза подряд для выхода
- Если "ПОКА!" не три раза подряд - бабушка отвечает случайным годом
- "ПОКА! ПОКА! ПОКА!" в одной строке не считается за три раза
